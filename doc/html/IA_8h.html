Implementa l'Intelligenza Artificiale dell'avversario.

Parametri
[in]manoMano del giocatore CPU
[out]idxIndice della carta scelta
[in]cCarta giocata dall'utente (NULL se CPU gioca per prima)
[in]briscolaSeme che determina la briscola nella partita

L'algoritmo che implementa l'Intelligenza Artificiale si basa sul dare un valore ad ogni possibile giocata, ovvero ad ogni carta, e di conseguenza giocare la carta migliore (più il valore assegnato è piccolo, migliore è la giocata). La funzione, dopo aver inizializzato alcune strutture dati di supporto, innanzituto guarda se la CPU è la prima a giocare o la seconda: se è la prima, applica sempre lo stesso criterio di valutazione, se è la seconda, prima valuta la carta c giocata precedentemente dal'utente (assegnandole un tipi_carta), quindi in base ad essa valuta le varie giocate.

Si veda anche
valuta_giocata_carico_briscola, valuta_giocata_carico, valuta_giocata_briscola e valuta_giocata_liscio per le varie scelte in base alla carta giocata c .
